# This is a simple expence tracker project using python 3 which lets you to add,show your dauly expence using csv file.this is a beginner friendly project learning file handling . 


## Features
- Add an expense with date, category, and amount
- View all expenses
- Stores data in a CSV file
- Beginner friendly project


## File Structure
- `expense_tracker.py` → Main program
- `expense.csv` → Expense data (auto-generated)
